# tfy-nats-ui

Helm chart for the NATS UI dashboard.

## Parameters

### Global parameters for Various Configurations

| Name                                          | Description                                                                                                                                                                                                                  | Value                                      |
| --------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
| `global.imagePullSecrets`                     | Existing truefoundry image pull secret name                                                                                                                                                                                  | `[]`                                       |
| `enabled`                                     | Deploy NATS UI Deployment and Service                                                                                                                                                                                        | `false`                                    |
| `nameOverride`                                | Override the component name used in labels and the Deployment name                                                                                                                                                           | `""`                                       |
| `fullnameOverride`                            | If set, overrides the Deployment full name (leave empty for <release>-tfy-nats-ui)                                                                                                                                           | `""`                                       |
| `replicaCount`                                | Number of replicas                                                                                                                                                                                                           | `1`                                        |
| `image.registry`                              | Image registry (optional)                                                                                                                                                                                                    | `tfy.jfrog.io`                             |
| `image.repository`                            | Container image repository (required when enabled)                                                                                                                                                                           | `tfy-private-images/nats-ui-dashboard`     |
| `image.tag`                                   | Image tag                                                                                                                                                                                                                    | `aae9ff49415bd8f7defc6259a6e21366834db4b4` |
| `imagePullPolicy`                             | Image pull policy                                                                                                                                                                                                            | `IfNotPresent`                             |
| `imagePullSecrets`                            | Image pull secrets                                                                                                                                                                                                           | `[]`                                       |
| `serviceAccount.create`                       | Create a dedicated ServiceAccount                                                                                                                                                                                            | `false`                                    |
| `serviceAccount.name`                         | Override ServiceAccount name (defaults to fullname when create is true; when create is false defaults to the cluster default unless set)                                                                                     | `""`                                       |
| `serviceAccount.annotations`                  | Service account annotations                                                                                                                                                                                                  | `{}`                                       |
| `serviceAccount.labels`                       | Extra labels for the ServiceAccount                                                                                                                                                                                          | `{}`                                       |
| `serviceAccount.automountServiceAccountToken` | Automount service account token into the pod (only used when create is true)                                                                                                                                                 | `false`                                    |
| `env`                                         | If non-empty, replaces natsUrl/accountSeedSecret wiring. Values are passed through tpl(); use ${k8s-secret/<secret>/<key>} or ${k8s-secret/<key>} with envSecretName (same pattern as truefoundry servicefoundryServer.env). | `{}`                                       |
| `envSecretName`                               | Secret name for shorthand ${k8s-secret/<key>} entries in env                                                                                                                                                                 | `""`                                       |
| `natsUrl`                                     | Value for NATS_URL when env is empty                                                                                                                                                                                         | `http://{{ .Release.Name }}-tfy-nats:4222` |
| `accountSeedSecret.name`                      | Existing Secret name when env is empty                                                                                                                                                                                       | `truefoundry-tfy-nats-secret`              |
| `accountSeedSecret.key`                       | Key inside the Secret                                                                                                                                                                                                        | `NATS_CONTROLPLANE_ACCOUNT_SEED`           |
| `service.name`                                | Kubernetes Service metadata.name (defaults to tfy-nats-ui)                                                                                                                                                                   | `tfy-nats-ui`                              |
| `service.type`                                | Service type                                                                                                                                                                                                                 | `ClusterIP`                                |
| `service.port`                                | Service port                                                                                                                                                                                                                 | `8080`                                     |
| `service.targetPort`                          | Container port (defaults to service.port)                                                                                                                                                                                    | `""`                                       |
| `service.annotations`                         | Service annotations                                                                                                                                                                                                          | `{}`                                       |
| `extraEnv`                                    | Extra environment variables for the container                                                                                                                                                                                | `[]`                                       |
| `resources`                                   | Container resources                                                                                                                                                                                                          | `{}`                                       |
| `podSecurityContext`                          | Pod security context                                                                                                                                                                                                         | `{}`                                       |
| `securityContext`                             | Container security context                                                                                                                                                                                                   | `{}`                                       |
| `nodeSelector`                                | Node selector                                                                                                                                                                                                                | `{}`                                       |
| `tolerations`                                 | Tolerations                                                                                                                                                                                                                  | `[]`                                       |
| `affinity`                                    | Affinity                                                                                                                                                                                                                     | `{}`                                       |
